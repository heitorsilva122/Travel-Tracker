# Travel Tracker

Travel Tracker é um projeto pessoal feito em Python para ajudar no controle financeiro de viagens internacionais.

O projeto foi inicialmente pensado para uma viagem ao Japão, permitindo registrar gastos em ienes e acompanhar conversões para reais e dólares. No futuro, a ideia é adaptar o programa para funcionar com outros países e moedas.

## Funcionalidades atuais

* Cadastro de itens de importação
* Cadastro de itens de consumo
* Pesquisa de itens
* Remoção de itens
* Edição de itens
* Verificação de saldo de importação
* Verificação do total gasto
* Verificação da cotação utilizada
* Salvamento e carregamento dos dados em arquivo JSON

## Tecnologias utilizadas

* Python
* JSON

## Status do projeto

O projeto está em desenvolvimento.

As cotações ainda são fixas, mas futuramente o projeto terá suporte a novas moedas e atualização automática por API.
